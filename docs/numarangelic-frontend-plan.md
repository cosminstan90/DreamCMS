# NumarAngelic Frontend Plan

## Direction
- Visual thesis: solar-spiritual, warm, luminous, ritual-like; less nocturnal than candvisam and more guidance-driven.
- Content plan: reassurance first, interpretation second, next steps third, conversion last.
- Interaction thesis: soft reveal-ins, light parallax on hero imagery, calm hover lift on guidance links and number clusters.

## Audience
- Users landing from Google on exact number sequences: 111, 222, 333, 444, 555, 777, 888, 999.
- Users looking for spiritual reassurance, synchronicity meaning, relationship/career/love guidance.
- Mostly mobile-first, quick-answer-first, but still open to longer guided reading.

## Global Design System
- Primary palette: warm ivory, sun gold, amber, terracotta accents.
- Typography: serif for spiritual headlines, clean sans for utility copy.
- Surfaces: fewer cards, more layered sections and editorial bands.
- Brand feel: modern spiritual publication, not generic numerology blog.

## Page Types

### 1. Homepage
- Role: strongest brand statement + entry to top number intents.
- Hero: dominant full-bleed section with big number sequence motif and short reassurance copy.
- Support sections:
  - Top sequences today
  - Life areas: love, twin flame, career, money, manifestation
  - Popular guides / recent interpretations
  - Newsletter / ritual signup
- CRO:
  - primary CTA to most popular number page
  - secondary CTA to search
  - native ad between support and recent guides

### 2. Category Listing
- Role: thematic hub for intents like love, twin flame, breakup, money, career.
- Layout: editorial hero + featured guide + supporting list + right rail with quick paths.
- CRO:
  - sticky rail with newsletter + top related sequences
  - one integrated sponsorship band after 3-4 entries

### 3. Number Detail Page
- Equivalent to symbol/detail page but for a number sequence.
- Hero:
  - giant number sequence
  - short spiritual definition
  - direct answer block for AI/search snippets
- Core sections:
  - general meaning
  - love meaning
  - twin flame meaning
  - career/money meaning
  - when you keep seeing it
  - what to do next
- CRO:
  - affiliate/recommendation block after explanation depth
  - newsletter framed as daily/weekly angel numbers digest
  - related sequences at bottom

### 4. Guide / Article Page
- Role: long-form explanatory content for broader queries.
- Structure:
  - reassurance-first intro
  - answer block
  - scannable H2 flow
  - action/ritual box mid-article
  - related sequences + cluster links
- Monetization:
  - premium integrated ad slot after first answer block
  - recommendations after mid-depth section

### 5. Hub / Pillar Page
- Role: topical authority page for "Numere angelice in dragoste", "Numere angelice si twin flame", "Semnificatia orelor in oglinda".
- Design:
  - strong editorial hero
  - cluster overview
  - support pages grouped by intent
  - trust/methodology section
- Monetization:
  - sponsor placement between cluster groups
  - newsletter CTA as "ritual digest"

### 6. Search Page
- Role: fast utility page with premium feel.
- Design:
  - big search-first interface
  - grouped results by number / guide / category
  - suggested popular searches when empty
- CRO:
  - one subtle newsletter panel after results

### 7. Authors Page
- Role: trust and E-E-A-T support.
- Design:
  - editorial grid with spiritual-expert framing
  - methodology summary up top
- CRO:
  - no aggressive ads; trust page first

### 8. Author Detail Page
- Role: deepen trust for YMYL-adjacent spiritual content.
- Include:
  - credentials / approach
  - recent interpretations
  - topic focus areas
- Monetization:
  - minimal, only subtle internal recirculation

### 9. Static Trust Pages
- Pages: About, Contact, Editorial Policy, Affiliate Disclosure, Privacy, Terms.
- Design:
  - simple premium layout
  - stronger trust and brand story
  - no ad-heavy treatment

## Reusable Sections
- Quick answer band
- Number sequence rail
- Life-area switcher
- Ritual / next steps box
- Reassurance quote block
- Newsletter ritual CTA
- Sponsored discovery band
- Related guides strip

## Monetization Model
- Ads should feel like calm editorial sponsorship, not banners dropped into content.
- Affiliate blocks should be framed as practical next steps or ritual companions.
- High-intent pages for monetization:
  - number detail pages
  - love / twin flame / breakup guides
  - manifestation / abundance content

## Implementation Order
1. Public shell + homepage
2. Number detail template
3. Guide/article template
4. Category/hub pages
5. Search page
6. Authors + trust pages
7. CRO polish + monetization placements

## Success Signals
- Higher pages/session from number detail to related sequences
- Better newsletter signup rate from organic traffic
- Higher affiliate CTR on practical recommendation blocks
- Better ad visibility without hurting perceived quality
